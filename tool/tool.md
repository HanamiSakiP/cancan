# 要继续设置吗？[y/N/q]

What is your email?

We need to unlock your newly created private key now! Please enter the passphrase you just generated.
Do you want to export your public key to "1.pub.key"? [y/N/q]:

# 请输入用于密码库 git 配置的电子邮件地址 []：
Please enter an email address for password store git config []:
# 您要添加一个 git 远程吗？[y/N/q]:
Do you want to add a git remote? [y/N/q]:
